# 🗺️ Mapa de Habilidades - Desafio DIO

Este repositório contém o resultado do desafio de projeto **"Identificando Habilidades que podem ser Remuneradas"**, promovido pela [DIO](https://www.dio.me/).

O objetivo deste projeto foi realizar um mapeamento pessoal de competências, identificar oportunidades de mercado e traçar estratégias para monetização, com o auxílio de ferramentas de Inteligência Artificial.

## 📋 Etapas do Desafio

O projeto foi dividido em três etapas principais, conforme o guia de atividades:

### 1. Inventário de Habilidades 🧠
Nesta etapa, o foco foi o autoconhecimento. Foram listadas competências baseadas em perguntas reflexivas, tais como:
- O que as pessoas costumam elogiar em mim?
- Em quais tarefas me sinto confiante?
- Que tipo de ajuda ofereço a amigos/familiares?
- O que faço com facilidade que outros acham difícil?

> **Uso da IA:** A Inteligência Artificial foi utilizada para ajudar a expandir a lista e correlacionar hobbies com habilidades profissionais.

### 2. Cruzamento com Necessidades do Mercado 📊
Aqui, as habilidades listadas foram analisadas sob a ótica comercial:
- Existe demanda para essa habilidade?
- É um problema comum que precisa de solução?
- Existem nichos específicos que valorizam isso?

> **Ferramentas:** Foram exploradas tendências usando Google Trends, LinkedIn e análise de mercado via IA.

### 3. Caminhos de Monetização 💰
Definição de como transformar as habilidades em renda. As estratégias consideradas incluem:
- **Plataformas de Freelance:** Workana, 99Freelas, Fiverr, GetNinjas.
- **Criação de Conteúdo:** Instagram, TikTok, YouTube, LinkedIn.
- **Portfólio:** Uso do GitHub para demonstrar projetos.
- **Educação:** Mentorias ou aulas.
- **Parcerias Locais:** Com pequenos negócios.

## 🛠️ Ferramentas Utilizadas

- **Inteligência Artificial:** ChatGPT / Microsoft Copilot (para brainstorming de habilidades, análise de mercado e criação de propostas/bios).
- **GitHub:** Hospedagem do portfólio e documentação.
- **Plataforma DIO:** Curso base para a metodologia.
